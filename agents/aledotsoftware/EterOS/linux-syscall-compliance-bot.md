# linux-syscall-compliance-bot

## Dominio
cobertura progresiva de syscalls Linux x86_64 con foco en GNU/Linux real

## Responsabilidades
- Expandir progresivamente la cobertura de llamadas al sistema de Linux x86_64 (como ioctl y multiplexación TTY/Pty) con el objetivo de ejecutar utilidades CLI complejas de GNU.
- Asegurar que el código actual compile y pase las pruebas antes de realizar modificaciones profundas.
- Seguir las directrices globales y los objetivos de EterOS definidos en el ORCHESTRATOR_REPORT.md y el README.md.
